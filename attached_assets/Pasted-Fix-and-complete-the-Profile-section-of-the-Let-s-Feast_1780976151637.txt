Fix and complete the Profile section of the Let’s Feast customer app.

Right now the Profile page shows menu items, but they are not fully clickable or usable.

Make every Profile menu item clickable and connected to a working page, modal, or screen.

PROFILE PAGE REQUIREMENTS:

The Profile page should show:

- User name
- Email
- Phone number
- Membership status
- Feast Rewards points
- Profile photo/avatar placeholder

Then show clickable menu cards for:

1. Payment Methods
2. Food Preferences
3. Order History
4. Feast Rewards
5. Settings
6. Help & Support
7. Log Out

Each row should:
- Be clickable
- Have a hover/tap effect
- Have a right arrow
- Navigate to the correct page
- Use modern mobile app styling

IMPORTANT:
Do not leave these as static list items.
Each one must actually open a functional screen.

- Saved card placeholder
